What is Ringlight?

Ringlight is a combination of peer-to-peer and Web 2.0 technologies designed to help you manage your files across multiple computers.


You can download or stream files between your various computers (home and work, desktop and laptop, server and media center, etc.) over the Internet. You can share your photos and videos with friends through both a local network (coffee shop sharing) and publically over the Internet (web publishing).


Ringlight does all this by combining a very small (2MB) client program running on each computer with a Web 2.0 website. The website allows you to manage all of your files easily from anywhere through your web browser. The client keeps track of changes to your collection of files and manages transferring those files peer-to-peer to other computers.


Unlike sites like Flickr and YouTube, Ringlight works with all of your files, not just one particular type. You never have to upload files and there are no file size limits. The files are served directly from your computer, so if you ever want to take them down, just move them out of the shared folder!


Managing your files, either for your own private use, or so that you can publish them to the world, has never been easier. The Ringlight website even has Flash media players for common types of content (mp3 audio files, mov video files, and many picture formats), so you can stream your files right in your browser without even needing to download them first.

Fun Facts

The Ringlight client is written in python with minimal library dependence so as to be small and cross-platform.


The Ringlight server consists of a part the client talks to written in python and SQLAlchemy and the website, which is written in Java using Momoko and Velocity.


The database backend is MySQL running on Debian GNU/Linux.


You can view the server stats if you'd like.
